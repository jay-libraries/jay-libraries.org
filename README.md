# Jay Libraries - Online Book Library

Welcome to **Jay Libraries**, your online platform for reading, publishing, and monetizing novels. Fully functional, dark theme, with premium book support, reading history, watermarks, and piracy reporting.

---

## **Features**

- Browse novels by **genre** or **search by title/author**
- **Premium books**: unlock via Monnify payments
- **Watermarks** to prevent unauthorized sharing
- **Reading history** tied to user accounts
- **Top contributors** dynamically displayed via JSON
- **Burger menu** with functional tabs: Account, Reading History, Publish, Top Contributors, How to Use, Report Piracy, About
- **Report piracy** sends email to admin (**jaylibraries@gmail.com**) and optionally auto-bans book in user UI
- Supports **PDF** and **Word documents**
- Fully responsive and mobile-friendly
